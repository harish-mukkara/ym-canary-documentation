app.global.staticProperties

Input Parameters#

None - Input will be automatically taken from app.source

Output Parameters#

A boolean response whether the static property checked is true or false

Function#

app.global.isWA = app.source == "whatsapp" ? true : false;
app.global.isYM = app.source == "yellowmessenger" ? true : false;
app.global.isGA = app.source == "googleassistant" ? true : false;
app.global.isVB = app.source == "voice" ? true : false;

Example#

Example 1#

app.global.isWA;

The above code will give the response as true if it is whatsapp and false if it is any channels other than whatsapp

Example 2#

app.global.isYM;

The above code will give the response as true if it is yellowmessenger and false if it is any channels other than yellowmessenger
